AI Based Applicant Tracking System

This project is a Flask-based web application that automates resume screening.

Features
- Add Job
- Upload Resume
- Extract Resume Text
- Match Skills with Job Description
- Calculate Match Score
- Rank Candidates

Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap
- SQLite
- PyPDF2

How to Run
1. Install dependencies
pip install flask PyPDF2

2. Run the project
python app.py

3. Open browser
http://127.0.0.1:5000
